SMART TRAVEL

AI-POWERED ECO ASSISTANT

THE PROJECT

Smart Travel Assistant is an intelligent application designed to modernize the travel experience. It provides personalized recommendations and real-time updates with a unique focus on Sustainability. The app helps users minimize their environmental impact by calculating carbon footprints and suggesting eco-friendly routes.

THE OBJECTIVE

To build a "Conscious Navigation" tool. The goal was to integrate multi-modal transportation logic (Bus vs. Train vs. Walking) to offer users optimal choices that balance efficiency with eco-responsibility.

THE TECH STACK

  • Python 3.8+ (Core Logic)
  • Flask (Web Framework)
  • OpenRouteService (Mapping API)
  • Geopy (Geolocation Data)

KEY LEARNINGS

This project taught me the intricacies of Geospatial Engineering. I learned how to work with coordinate systems using Geopy and how to consume complex external APIs (OpenRouteService) to render dynamic maps and calculate real-time distances programmatically.

CHALLENGES & SOLUTIONS

The Challenge: Data Normalization. The API data for different transport modes (e.g., public transit vs. driving) came in different formats, making it hard to compare them fairly for the "Eco-Score."

The Solution: I built a custom Python adapter class that standardizes the incoming API responses, allowing the application to mathematically compare the carbon footprint of a bus ride versus a car ride instantly.
