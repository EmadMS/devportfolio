CYBER DUST

IMMERSIVE 3D WEB EXPERIENCE

THE OBJECTIVE

To build a portfolio piece that pushes the boundaries of WebGL in the browser. The goal was to create a particle system that reacts to mouse movement without lagging on mobile devices.

THE TECH STACK

  • React.js Framework
  • Three.js / React-Fiber
  • GSAP for timeline animations
  • GLSL Shaders

CHALLENGES

Optimizing the particle count was difficult. I had to use instanced meshes to keep the draw calls low, ensuring a steady 60FPS.
